On-device OREF risk models (optional)
=====================================

Place these files here to enable local ML scores in AIMI Advisor (no cloud, no Nightscout):

  hypo_lgbm.onnx
  hyper_lgbm.onnx
  bg_change_lgbm.onnx

Input: float32 [N, 35] with feature order matching OrefModelFeatures.NAMES in code
(and models/model_meta.json "features" when you train against that manifest).

Stubs (neutral scores, not clinical): regenerate with
  python scripts/generate_oref_stub_onnx.py
(requires: pip install onnx; IR version is set for onnxruntime-android compatibility.)

Real models: export from LightGBM (.pkl) with skl2onnx / onnxmltools; use the same
35-feature order and shapes:
  hypo/hyper: output [N, 2] class probabilities
  bg_change: output [N, 1] regression

Rebuild the app after replacing files. If files are absent, Advisor still shows metrics + heuristics.
